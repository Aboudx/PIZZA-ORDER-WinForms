# 🍕 Pizza Order System

A simple educational project built with **C# Windows Forms** to practice using common GUI controls and handling user interactions.

## About

This application allows users to create a pizza order by selecting:

- Pizza size
- Toppings
- Crust type
- Quantity
- Dining option (Eat In / Take Out)

The application then displays an order summary and calculates the total price.
